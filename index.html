Портфолио TheGamerDog

Seolity

SEO-анализатор с бесплатным тарифом, направленный на улучшение выдачи сайта в поисковых системах.

В проекте используется React в качестве фреймворка для Frontend, express для Backend. Занимался созданием текстового анализатора страницы с вычислением Tf-idf, количества слов в тексте и в ссылках, ширины, глубины, релевантности. Кроме того, занимался исправлением багов, переписывал старые модули проекта под новый функционал.

Экзаменатор

Админ-панель для бота в Telegram для опроса пользователей

В проекте используется Next в качестве фреймворка для Frontend, NestJS для Backend. Занимался созданием админ-панели для управления вопросами и категориями вопросов, помогал с созданием самого бота

Рандомайзер-бот

Бот Telegram для выбора победителя в конкурсе, веб-приложение с анимацией для определения победителя

В проекте используется чистый JS для веб-приложения, NestJS для бота. Занимался разработкой функционала для бота, для веб-приложения создал анимацию прокрутки

Игра на английском

Серверная часть игры для обучения английскому языку

В проекте используется PHP в качестве фреймворка для серверной части. Занимался разработкой авторизации с подтверждением по почте, менеджером ивентаря пользователя и прочих данных, наград за определенные действия с проверками

User-бот панель Discord

Панель для управления юзерботами в Discord

В проекте используется React, WebSocket в качестве связки сервера с клиентом. Разработал панель, в которой можно наблюдать за сообщениями (получать уведомления о их приходе, при подписке на канал), расставлять реакции на сообщения, возможность менять токены (или использовать несколько разом)
